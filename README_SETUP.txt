
IIAOC Tools — One-Page Setup (GitHub Pages)

WHAT TO USE
- This folder contains everything you need to publish your tools:
  /index.html
  /tools/ach.html
  /tools/rcs.html
  /tools/em-imint-tasks.html

STEP 1 — Create an empty GitHub repo (public)
- Name it, e.g., iiaoc-tools
- Leave it empty or with a README; both are fine.

STEP 2 — Upload these files
- Click Add file → Upload files
- Upload the entire 'iiaoc-tools' folder contents so your repo looks like:
    /index.html
    /tools/ach.html
    /tools/rcs.html
    /tools/em-imint-tasks.html

STEP 3 — Enable GitHub Pages
- Repo → Settings → Pages
- Source: Deploy from a branch
- Branch: main • /(root)  → Save
- Wait for the green banner with your URL:
  https://<username>.github.io/<repo>/

STEP 4 — Enforce HTTPS
- On the same Pages screen, ensure “Enforce HTTPS” is checked.

STEP 5 — Share the URL
- Students open the homepage and click the tool buttons.

OPTIONAL — Custom domain
- Settings → Pages → Custom domain (e.g., tools.example.com)
- Add the shown CNAME at your registrar; then enforce HTTPS.
